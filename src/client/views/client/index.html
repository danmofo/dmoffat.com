My name is Daniel and I'm a website developer from the West Midlands.

I design and build performant, aesthetically-pleasing websites and web applications, from the conception stage all the way through to the build.

Experience and skills

Experience back to top

I’ve been a web developer professionally for 6 years, currently working in-house at Give as you Live, a fundraising platform that enables users to raise money for a charity when shopping online.

I do a mixture of frontend and backend development as part of my day-to-day job, but don’t design, however I do understand and appreciate what “good” looks like and enjoy designing my own personal projects.

Along with my commercial experience, in my spare time I frequently experiment and build new things using tech I don't get to use at my day job.

Technical skills

I try to frequently gain exposure to as many technologies and software as possible, quickly adapting to learn new things. If something achieves a task in a more efficient manner, I’ll gladly swap to it. Here's a list of things I know, it's by no means extensive (for that, you can check out my CV), I have not listed anything I’m not comfortable using, so you won’t see things I haven’t used for a large amount of time, even though I may have had exposure to it in the past.

Programming Languages

I am comfortable programming in Java for the server-side (using things such as Maven, Spring 4, Hibernate.) & JavaScript for the client-side (using things such as NPM & jQuery).

JavaScript frameworks

I’ve built a few applications at work using AngularJS 1.x and have used Vue 2 in several personal projects, making use of Vue Router and VueX.

Task managers & tooling

I use Grunt daily, making use of Browserify for module bundling and a range of plugins for optimising static assets.

"Styling"

Extensive experience writing modular and reusable CSS, using the preprocessors SASS and LESS.

Data storage

I’ve used both MySQL and Elasticsearch professionally for a few years.

Design software

I use Sketch for my designs and creating all new assets. I also have a lot of experience using Adobe Photoshop when required for designing & exporting assets.

Operating systems

I’m primarily a OSX user, but have a lot of experience using Windows as my secondary machine. I know enough about using Linux (Ubuntu) to be deadly!

Version Control Systems

Strong working knowledge of both Git and SVN. I am comfortable creating, maintaining and using repositories of this type.

IDE / Text editing software

I use Eclipse daily for Java development at work. IntelliJ for all new personal Java projects. Sublime text 3 and a huge amount of plugins for everyday tasks.

Misc

I am comfortable using and configuring an Apache Web Server. I test all of my code using JUnit and Karma / Jasmine.

Templating / view technologies

I know HTML(5), Velocity, Freemarker, Handlebars, EJS, Jinja2, XSLT and any other templating language you can think of - they usually have different syntax but implement the same logic.
